# Remote Sensing for Groundwater Change Detection in Rajasthan

This project analyzes groundwater stress in Rajasthan using satellite-derived
groundwater indicators and machine learning techniques.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Machine Learning Model
- Random Forest Regressor

## Dataset
Open-source groundwater dataset exported from Kaggle.

## Results
The model successfully identifies groundwater-stressed districts in Rajasthan
based on recharge and extraction indicators.
